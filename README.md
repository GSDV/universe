## Setup
Run
```bash
 git clone https://github.com/GSDV/universe.git
 npm install
```
to get the code.

In the ```.env``` file, add ```EXPO_PUBLIC_DOMAIN``` variable to point to your local development url:
```
EXPO_PUBLIC_DOMAIN=http://localhost:3000
```
